# AI-Career-Recommendation-Platform
Full-stack AI career recommendation platform for VCE students with Python, Anvil, Gemini 2.5 Pro API integration, and custom HTML/CSS frontend.


# Future Ready – AI Career Recommendation Engine
Platform: Anvil (Full-stack Python Framework)  
AI Model: Gemini Pro 2.5 
Frontend: HTML & CSS  
Backend: Python & Anvil Server Functions  
Data: Multi-table Database Architecture  

---

## Overview
Future Ready is a full-stack AI web application designed to guide Victorian secondary students in career planning.  
Built using Anvil, Python, and HTML/CSS, it provides a responsive interface with AI-driven recommendations, actionable roadmaps, task management, and analytics.  
The system focuses on moving students from career discovery to execution.

---

## Key Features

### Dynamic Multi-Page Website
- Structured pages: Home, Mission, Support.  
- Static navigation bar and side menu for seamless transitions.  
- Multi-page functionality achieved through Anvil’s modular form hierarchy.  
- Custom HTML/CSS for styling, following clear naming and documentation conventions.  

### Signup & Login System
- Independent authentication linked to Anvil Data Tables.  
- Tailored error handling to ensure robustness.  
- Tested across multiple browsers for reliability.  

### AI Career Recommendation Engine
- Integrated Gemini 2.5 Pro model, tuned via Google AI Studio.  
- Users input hobbies, interests, subjects, and career goals.  
- AI outputs:
  - Career recommendations  
  - Rationale for each suggestion  
  - Alternative options  
  - Required VCE subjects  
  - Additional certifications  
  - Relevant tertiary pathways  
- Contextual memory: previous user interactions are stored and referenced for consistent advice.

### Interactive Career Roadmap
- Auto-generated roadmap based on user profile and AI advice.  
- Users can add, delete, or regenerate steps.  
- Implemented with Anvil repeating panels and string-parsing mechanisms.  
- Modifications persist across sessions in the database.  

### Productivity & Task Management System
- Users can create, update, and track tasks with multiple attributes.  
- Tasks assigned a value score to prioritise completion.  
- Server-side functions maintain performance and reliability.  

### Analytics Dashboard
- Real-time graphs powered by Plotly visualise task progress and motivation.  
- Displays open, completed, and total tasks alongside current goals.
